# 간단 출력 프로그램

### Development environment

- Html + Css + Bootstrap + JavaScript
- node.js + Express.js

### Features


1. 이름 및 학번 기본 정보 출력
2. form의 입력창에 내용 입력 후 전송 버튼을 누르면 페이지를 새로 고치면서 적은 내용이 출력
